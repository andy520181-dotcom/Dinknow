### 第二步：创建数据库集合

#### 2.1 打开云开发控制台
```bash
云开发 → 数据库 → 添加集合
```

#### 2.2 创建以下集合

##### 集合 1：`users` - 用户信息
```json
{
  "_id": "用户ID（自动生成）",
  "openid": "微信openid",
  "nickName": "昵称",
  "avatarUrl": "头像URL",
  "gender": "性别（0未知/1男/2女）",
  "duprLevel": "DUPR水平（2.0-5.5）",
  "createdAt": "创建时间（时间戳）",
  "updatedAt": "更新时间（时间戳）"
}
```

**权限设置**：
- 读权限：仅创建者可读写
- 写权限：仅创建者可读写

##### 集合 2：`activities` - 活动信息
```json
{
  "_id": "活动ID（自动生成）",
  "title": "活动主题",
  "startDate": "开始日期（YYYY-MM-DD）",
  "startTime": "开始时间（HH:mm）",
  "address": "球场地址",
  "latitude": "纬度",
  "longitude": "经度",
  "maxParticipants": "最大参与人数",
  "fee": "活动费用",
  "description": "活动描述（可选）",
  "duprLevel": "DUPR要求（可选）",
  "hostId": "发起人openid",
  "hostName": "发起人昵称",
  "hostAvatar": "发起人头像",
  "status": "状态（pending/ongoing/completed/cancelled）",
  "createdAt": "创建时间（时间戳）"
}
```

**权限设置**：
- 读权限：所有用户可读
- 写权限：仅创建者可读写

##### 集合 3：`registrations` - 活动报名记录
```json
{
  "_id": "报名ID（自动生成）",
  "activityId": "活动ID",
  "userId": "用户openid",
  "userName": "用户昵称",
  "userAvatar": "用户头像",
  "status": "状态（joined/cancelled）",
  "joinedAt": "报名时间（时间戳）"
}
```

**权限设置**：
- 读权限：所有用户可读
- 写权限：仅创建者可读写

#### 2.3 创建索引（可选，提升性能）

**activities 集合索引**：
```bash
字段名: createdAt
类型: 降序
```

**registrations 集合索引**：
```bash
字段名: activityId
类型: 升序
```

## 🔧 常见问题

### 问题1：云函数调用失败
**错误**：`cloud.callFunction:fail`

**解决方案**：
1. 确认云函数已部署
2. 确认云开发环境ID正确
3. 检查云函数权限设置
4. 查看云函数日志排查错误

### 问题2：数据库读写失败
**错误**：`database permission denied`

**解决方案**：
1. 检查数据库权限设置
2. 确认云函数有数据库读写权限
3. 检查集合名称是否正确

### 问题3：位置授权失败
**错误**：`getLocation:fail`

**解决方案**：
1. 确认隐私保护指引已审核通过
2. 确认 `__usePrivacyCheck__: true` 已配置
3. 确认 `requiredPrivateInfos` 已声明 `getLocation`
4. 使用真机测试

### 问题4：图片上传失败
**错误**：`uploadFile:fail`

**解决方案**：
1. 确认云存储已开通
2. 创建 `avatars/` 文件夹
3. 检查云存储权限设置
