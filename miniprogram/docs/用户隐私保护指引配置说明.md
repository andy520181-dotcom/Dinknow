# 微信小程序「用户隐私保护指引」配置说明

## 一、配置入口

1. 登录 **微信公众平台**：https://mp.weixin.qq.com  
2. 进入你的小程序（若有多主体，先选对应小程序）  
3. 任选以下一种方式进入隐私指引配置：
   - **方式 A**：左侧菜单 **设置** → **基本设置** → 页面中找到 **服务内容声明** → 点击 **用户隐私保护指引**
   - **方式 B**：左下角 **头像/账号** → **账号设置** → **服务内容声明** → **用户隐私保护指引**

> 若未看到「用户隐私保护指引」入口，可先在 **设置 → 基本设置** 中查看是否有「服务内容声明」或「隐私设置」相关入口；不同账号类型/时间可能略有差异。

---

## 二、需要声明的信息（与本项目相关）

你的小程序使用了以下能力，需在「**开发者处理的信息**」中**勾选并填写用途**：

| 处理的信息 | 对应接口/能力 | 建议用途说明 |
|------------|----------------|--------------|
| **收集你选中的照片或视频信息** | wx.chooseImage / wx.chooseMedia / wx.chooseVideo | 用于用户选择或拍摄头像、更换个人头像 |
| **收集你的位置信息** | wx.getLocation 等 | 用于显示附近匹克球活动与计算距离（与 manifest 中 scope.userLocation 一致） |
| **收集你选择的位置信息** | wx.chooseLocation | 用于发起活动时选择活动地点 |

- 微信会根据你**实际调用的隐私接口**在后台展示**必填项**，请至少把上述几项勾选并填好用途。  
- 若还使用了**摄像头**（拍照）、**相册**等，可能还会出现「访问你的摄像头」等项，按实际使用勾选并说明用途即可。

---

## 三、操作步骤（简要）

1. **进入「用户隐私保护指引」设置页**（见「一、配置入口」）。
2. **编辑「开发者处理的信息」**  
   - 找到并勾选：**「收集你选中的照片或视频信息」**。  
   - 在对应项下填写**处理目的**，例如：  
     - `用于用户选择或拍摄头像、更换个人头像。`
3. **位置相关**（若后台有展示）  
   - 勾选 **「收集你的位置信息」**，用途可填：`用于显示附近匹克球活动与计算距离。`  
   - 勾选 **「收集你选择的位置信息」**，用途可填：`用于发起活动时选择活动地点。`
4. **其他必填项**  
   - 按页面提示完成：引导语、用户权益、联系方式、存储期限、使用规则等（一般有示例可参考）。  
5. **保存并提交**  
   - 保存后，需**重新提交代码审核并发布**，隐私指引才会在线上生效。  
   - 若为第三方代开发，需由第三方通过接口配置并随版本一起提审。

---

## 四、注意事项

- **与代码一致**：声明的信息类型必须与小程序实际调用的接口一致（如已用 chooseMedia 选图，就必须声明「收集你选中的照片或视频信息」），否则可能审核不通过或真机无法调起相册/拍照。  
- **生效时间**：配置保存后，通常需**重新提审并发布**才会在正式版/体验版生效；开发版有时需重新编译或真机调试一次。  
- **相册/拍照失败**：若已按上文声明「收集你选中的照片或视频信息」并发布，仍无法选图或拍照，可再检查：  
  - 真机是否授权了相册/摄像头；  
  - 微信与系统设置中是否禁止了该小程序的相册/相机权限。
