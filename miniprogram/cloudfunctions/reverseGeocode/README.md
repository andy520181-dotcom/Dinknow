# 逆地理编码云函数

用于根据经纬度自动获取地址文案，并在广场左上角显示，**不再弹出“选择位置”界面**。

## 配置步骤

1. 登录 [腾讯位置服务](https://lbs.qq.com/console/mykey.html) 申请 WebService Key。
2. 在微信云开发控制台 → 当前环境 → 云函数 → `reverseGeocode` → 配置 → 环境变量中新增：
   - 键：`TENCENT_MAP_KEY`
   - 值：上述 Key
3. 保存并重新部署该云函数。

配置完成后，用户允许位置授权即可自动得到地址并显示在广场左上角；未配置时左上角会显示「已定位」或坐标。
