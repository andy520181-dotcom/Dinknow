# 清除测试活动数据指南

## 方法一：使用云函数（推荐）

### 1. 部署清理云函数
1. 在微信开发者工具中，找到 `cloudfunctions/cleanupTestData` 文件夹
2. 右键点击该文件夹
3. 选择「上传并部署：云端安装依赖」
4. 等待部署完成

### 2. 调用云函数
在微信开发者工具的「云开发」控制台中：
1. 进入「云函数」页面
2. 找到 `cleanupTestData` 函数
3. 点击「测试」按钮
4. 输入测试参数：`{}`
5. 点击「运行」执行清理

或者在代码中调用：
```javascript
wx.cloud.callFunction({
  name: 'cleanupTestData',
  data: {}
}).then(res => {
  console.log('清理结果:', res.result)
})
```

## 方法二：通过云开发控制台手动删除

### 1. 删除报名记录
1. 打开微信开发者工具 → 「云开发」→ 「数据库」
2. 选择 `registrations` 集合
3. 点击「管理」→ 全选所有记录 → 「删除」

### 2. 删除活动记录
1. 在「数据库」中选择 `activities` 集合
2. 点击「管理」→ 全选所有记录 → 「删除」

## 注意事项

⚠️ **警告**：此操作会永久删除所有活动数据和报名记录，无法恢复！

- 建议只在测试环境使用
- 生产环境请谨慎操作
- 如需保留部分数据，请先导出备份

## 清理后的效果

- ✅ 所有活动（activities）被删除
- ✅ 所有报名记录（registrations）被删除
- ✅ 用户数据（users）不受影响
